常见控件属性详解

发表于 2017-12-24 | 分类于 UI

一、控件介绍

1.1 文本类控件

1.1.1 TextView

 用于向用户显示文本内容，不可编辑。

1.1.2 EditText

 用于向用户显示文本内容，可编辑。

1.2 按钮类控件

1.2.1 Button

 用户可点击的按钮。

1.2.2 CheckBox

 复选/多选按钮，支持选中/取消选中。

1.2.3 RadioButton

 单选框双状态按钮，支持单击选中，不支持单击取消选中。

1.3 图像类控件

1.3.1 ImageView

 图片控件用于显示图片。

1.3.2 ImageButton

 图片和按钮结合的产物。

1.4 进度类控件

1.4.1 ProgressBar

 进度条，用于显示当前进度。

1.4.2 SeekBar

 滑动进度条，用于滑动设置进度。

二、控件共有属性

2.1 id

  用于定义指定控件对应的ID。
android:id=”@+id/test_id” // ‘+’创建id
  

2.2 width & minWidth & maxHeight

 用于定义指定控件的宽度/最小宽度/最大宽度，单位有px,dp.sp等。

2.3 height & minHeight & maxHeight

 用于定义指定控件的高度/最小高度/最大高度，单位有px,dp.sp等。

2.4 background

 用于定义指定控件的背景(色)，颜色采用RGB命名法。

2.5 clickable & focusable

 用于定义指定控件是否可点击/有焦点。

2.6 alpha

 用于定义指定控件的透明度。

2.7 visibility

 用于定义控件是否可见。

三、控件特有属性

3.1 text

 用于定义文本/按钮类控件的文字内容。

3.2 textSize

 用于定义文本/按钮类控件的文字大小，单位有px,dp.sp等。

3.3 textColor

 用于定义文本/按钮类控件的文字颜色，RGB命名法。

3.4 textAllCaps

 用于定义文本/按钮类控件的文字是否全部大写。

3.5 lines & minLines & maxLines & sigleLine

 用于定义文本/按钮类控件显示的文字行数/最小行数/最大行数/单行显示。

3.6 minLength & maxLength

 用于定义文本/按钮类控件显示的文字最小/最大长度。

3.7 scaleType

 用于定义图像类控件中图像资源的显示方式。

3.8 textStyle

 用于设置字体形状。

3.9 hint

 用于定义文本/按钮类控件的文本为空时显示的提示型文字。

3.10 inputType

 用于定义文本/按钮类控件的文本类型。

3.11 src

 用于定义图像类控件的图像资源。

3.12 progree

 用于定义进度类控件的默认进度值。

3.13 autoLink

 用于定义文本/按钮类控件的中哪类文本自动设为链接。
