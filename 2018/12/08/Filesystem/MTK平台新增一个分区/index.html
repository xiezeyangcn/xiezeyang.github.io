MTK 平台新增一个分区

发表于 2018-12-08 | 分类于 Filesystem

下面将以MT6765平台新增oem partition为例：
(更多请参考https://android-review.googlesource.com/q/topic:product_partition+(status:open+OR+status:merged))

一、device/mediatek目录

1.1 mt6765/BoardConfig.mk

1.2 mt6765/partition_table_MT6765.xls

在分区表中新增oem分区，其中：
Partition Name: 分区名称
Type: 分区类型
Size_KB：分区大小
Region：memory区域，默认EMMC_USER
Reserved：是否预置，默认N
Download：是否支持flashtool烧录
Download_File：分区对应的image文件
OTA_Update：是否支持OTA升级
EmptyBoot_Needed：是否空启动，默认N
FastBoot_Erase：是否支持fastboot擦除
Fastboot_Download：是否支持fastboot下载
Operation_Type：操作类型，默认AUTO

Partition_Name Type Size_KB Region Reserved Download Download_File OTA_Update EmptyBoot_Needed FastBoot_Erase FastBoot_Download Operation_Type
